# WINE REVIEWS
Проект посвящен анализу винных отзывов. 
Задача состоит в предсказании сорта винограда по тексту отзыва на вино.
Отзывы были собраны с помощью скраппера https://github.com/zackthoutt/wine-deep-learning/blob/master/scrape-winemag.py
Файлы red-varieties.txt white-varieties.txt содержат классификацию сортов винограда из Википедии.
